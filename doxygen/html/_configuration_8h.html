bool parse_commandline ( int32_t  argc,
char **  argv 
)

Parses the application command line, setting runtime configuration options. Separate function from the Configuration as it should only be executed once, and within app_init(), where the parameters are available.

Parameters
argcThe number of arguments passed in
argvAn array of pointers to the arguments
Returns
true is returned if any and all command line options are processed successfully
false is only returned if an unknown parameter is provided, or an invalid option is passed to a parameter
