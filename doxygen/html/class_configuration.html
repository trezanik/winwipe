Detailed Description

Holds all of the configuration variables for the application, and provides the ability to load from file, or retain the defaults.

void Configuration::Dump ( ) const

Dumps the entire configuration, as-is, to the application log file. LL_Debug does not need to be set, as it will be output directly, not checking the level.

void Configuration::LoadDefaults ( )

Loads the default configuration options. Should be called first, before Load().

The settings set here must constrast with those processed in the parse_commandline() function, as they are designed to override anything set here.
