\subsection{Detailed Description}
A two-\/dimensional grid, with a minimum of 2x2 -\/ each \hyperlink{class_tile}{Tile} must have a sibling so no \hyperlink{class_grid}{Grid} can have just one tile (minimum 4, if you can't work that out!). Outside of this restriction, grids can be any size, although a 3x100 grid wouldn't be much fun to play on unless you have a peculiar scenario.

While the \hyperlink{class_grid}{Grid} is not an ingame entity, it provides the foundation for the tiles to reside on.
