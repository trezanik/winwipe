\subsection{Detailed Description}
A spectator is always a human player, and as we want them to be able to browse a \hyperlink{class_level}{Level} unrestricted -\/ but not modify anything -\/ we inherit from a \hyperlink{class_player_controller}{Player\-Controller} and use an externally locked down class.

There can be up to 16 spectators in a \hyperlink{class_level}{Level}.
