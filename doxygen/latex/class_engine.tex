\subsection{Detailed Description}
Resides within the application runtime, and is the main bulk processing of the application. All game objects are stored here, where they can then be manipulated.

All classes inheriting from \hyperlink{class_object}{Object} are to be accessible from this \hyperlink{class_engine}{Engine} class -\/ if not, it is something that needs fixing.
