\subsection{Detailed Description}
A \hyperlink{class_player}{Player} is a member of a \hyperlink{class_team}{Team}, and has an \hyperlink{class_a_i_controller}{A\-I\-Controller} if non-\/human, or a \hyperlink{class_player_controller}{Player\-Controller} if human. If the \hyperlink{class_player}{Player} is a Spectator, they are provided only with a \hyperlink{class_spectator_controller}{Spectator\-Controller}, and are not allowed to interact with the \hyperlink{class_level}{Level}.
